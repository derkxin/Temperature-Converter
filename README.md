# Temperature-Converter

A simple Temperature Converter web-application

This is a simple web-based temperature converter application which develope by Java Spring Boot RESTful API, Javascript, HTML, CSS and Bootstrap5. It is a web-based application that allow users to convert the temperature between Celsius (°C), Fahrenheit (°F) and Kelvin (K). Besides, Java Spring Boot RESTful API is used to render the HTML file. Javascript is used to calculate the conversion of the temperature. HTML, CSS and Bootstrap5 are used to structure and design the webpage.

# To run the application:
1. Download the repository.
2. Open the folder in Eclipse IDE and run as java application.
3. Run localhost via browser and the interface should appear.
